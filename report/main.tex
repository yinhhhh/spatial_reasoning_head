\begin{abstract}
Spatial reasoning has remained a key challenge for Vision-Language
Models for a long time. We present a head-level analysis of spatial
reasoning and use the findings to design a stronger attention
intervention strategy. First, we analyze layer-17 head behavior with
YOLO-grounded overlap and overlap-to-correctness AUROC on
\texttt{Controlled\_Images\_A}, and find a strong negative relation
between ``spatially concentrated'' heads and ``functionally useful''
heads. This suggests that final correctness is often supported by heads
with weaker current spatial focus but higher latent functional
potential. Based on this, we introduce margin-guided continuous
attention reallocation, an improvement of AdaptVis that controls
intervention strength by decoding confidence margin and redistributes
image attention without hard clipping. Evaluation on four benchmarks
(\texttt{COCO\_QA\_one\_obj}, \texttt{COCO\_QA\_two\_obj},
\texttt{VG\_QA\_one\_obj}, \texttt{VG\_QA\_two\_obj}) shows consistent
improvements, especially on two-object settings.
\url{https://github.com/yinhhhh/spatial_reasoning_head}.
\end{abstract}

\section{Introduction}
Vision-language models (VLMs) have achieved strong performance across
multimodal tasks, yet spatial
reasoning remains fragile, especially when fine-grained object
relations must be inferred from visual evidence. A common line of work
addresses this issue by adjusting image-side attention during decoding,
with the goal of improving visual grounding and reducing relation
errors \citep{chen2024adaptvis}. While such attention reweighting methods are
increasingly used, most studies treat attention heads as a homogeneous
resource and focus on aggregate intervention strength.

We argue that this head-agnostic view misses an important mechanistic
question: \emph{do different heads contribute unequally to spatial
reasoning?} If so, intervention should not only decide \emph{how much}
visual attention to allocate, but also \emph{where in head space} this
allocation should be concentrated. 

This perspective motivates our core research question: 

\begin{quote}
  Can we characterize head-level functional differences in
  spatial reasoning, and leverage them to do attention reallocation
  to improve spatial reasoning performance?
\end{quote}
To answer this question, our project follows a three-stage narrative:
\begin{enumerate}[leftmargin=1.2em]
  \item \textbf{Head analysis:} establish that head-level spatial roles
  are real and measurable, and identify which heads are already
  spatially effective.
  \item \textbf{Improved allocation method:} design a better strategy
  for distributing image attention, moving beyond naive or purely global
  reweighting.
  \item \textbf{Mechanism-backed validation:} use the intervention
  outcomes to show that heads with latent but underutilized
  spatial-attention potential are key targets for improving spatial
  reasoning.
\end{enumerate}

\section{Preliminaries}
\textbf{Attention.}
Transformer attention maps an input token sequence to contextualized
representations through query, key, and value projections
\citep{vaswani2017attention}. Given token features
$X\in\mathbb{R}^{n\times d}$, each head computes
\begin{equation}
Q = XW_Q,\quad K = XW_K,\quad V = XW_V,
\end{equation}
and attention weights
\begin{equation}
A = \mathrm{softmax}\!\left(\frac{QK^\top}{\sqrt{d_k}}\right),
\end{equation}
followed by head output
\begin{equation}
Z = AV.
\end{equation}
Multi-head attention concatenates head outputs and applies a final
linear projection.

In autoregressive decoding, the model produces next-token logits $z_t$
at step $t$, and token probabilities are obtained by
\begin{equation}
p(y_t=i\mid y_{<t},x)=\frac{\exp(z_{t,i})}{\sum_j \exp(z_{t,j})}.
\end{equation}
The generated token is selected from this distribution (e.g., greedy or
sampling decoding).

\textbf{Vision-Language Models (VLMs).}
Models such as LLaVA and Qwen-VL
\citep{liu2023visual,bai2023qwen} process an image by first encoding it
with a vision backbone, e.g., ViT, which divides the image
into patches and converts them into a sequence of visual tokens.
These visual tokens are then projected into the
language model hidden space through a connector/projection module.
Finally, the visual tokens are concatenated with text tokens 
and fed into the decoder, so self-attention can jointly model
image-text interactions in one token sequence.

\section{Head-Level Spatial Analysis}
\noindent To figure out how different heads contribute
 to spatial reasoning, we first identify which heads are
spatially active and which heads are functionally tied to correctness.

\textbf{Experiment settings.}
We conduct the head-level analysis on \texttt{Controlled\_Images\_A}, a
synthetic and tightly controlled benchmark used in AdaptVis-style
evaluation. The images are human-constructed to isolate spatial
relations (e.g., left/right/on/under) with limited visual confounders,
which makes it suitable for diagnosing whether attention heads truly
capture spatial cues. The question set follows the AdaptVis protocol
\citep{chen2024adaptvis}: GPT-4-generated relation questions in a fixed-choice
format, such as ``Where is the object in relation to the anchor
object?'' with options \{left, right, on, under\}. In the AdaptVis
analysis, layer 17 attention is reported as most correlated with
generation behavior; therefore, we focus our head-level analysis on
layer 17.

\subsection{Head-wise attention distribution}

Visualizing attention maps is a standard way to inspect how a model
allocates visual focus. We first present head-wise attention heatmaps
from the no-intervention LLaVA-1.5 run.
Figure~\ref{fig:head32} (left) shows the 32-head panel on
\texttt{Controlled\_Images\_A} sample 0, revealing clear cross-head
heterogeneity in spatial focus.
The result is visually clear: some heads concentrate attention around
the core object (the sofa), while others do not. We tested multiple
samples and observed that this head-wise distribution pattern remains
largely consistent.

To quantify this phenomenon, we measure where each head places attention
mass using YOLO-grounded overlap. For each sample, we run YOLO
\citep{redmon2016you} to obtain object boxes and use a single-box
protocol for quantitative scoring to reduce ambiguity. The box is expanded by a fixed scale
(1.2$\times$ in our current setting) and rasterized into a binary mask
$M\in\{0,1\}^{H\times W}$. For each head $h$, we extract its attention
map, resize it to the same grid, and normalize it into a heatmap $H_h$
such that $\sum_{i,j} H_h(i,j)=1$. We then compute
\begin{equation}
\mathrm{Overlap}_h=\frac{\sum_{i,j} H_h(i,j)\,M(i,j)}{\sum_{i,j} H_h(i,j)}.
\end{equation}
Intuitively, this value is the fraction of a head's attention mass that
falls inside the detector-supported object region.
Figure~\ref{fig:head32} (right) shows a concrete single-head example
(head 2) with YOLO two-box overlay, illustrating how the overlap region
is grounded in detector output.

\subsection{AUROC-based functional ranking}
While overlap between attention distribution and YOLO detection reflects how much a head focuses on the object region,
overlap alone does not tell us whether that head is functionally useful
for the final answer.
We therefore evaluate each head by AUROC, using per-sample overlap
scores as the predictor and answer correctness ($0/1$) as the label.
Concretely, for each head $h$, we collect
$\{\mathrm{Overlap}_h(x), y(x)\}_{x\in\mathcal{D}}$, where $y(x)=1$
indicates a correct answer, and compute
\begin{equation}
\mathrm{AUROC}_h = \mathrm{AUC}\big(\mathrm{Overlap}_h(x)\rightarrow y(x)\big).
\end{equation}
Intuitively, AUROC measures how well overlap ranks correct samples above
incorrect samples \citep{fawcett2006introduction}. A larger AUROC means
the head's overlap is a stronger predictor of correctness; an AUROC near
0.5 indicates weak discriminative value.
Top AUROC heads are \{14, 29, 26, 6, 7, 4, 20, 17\}, which is almost disjoint from the top-overlap group.

This leads to a significant result:
\textbf{overlap and AUROC are strongly negatively correlated across
heads} (Pearson $r\approx-0.915$), as shown in
Figures~\ref{fig:inverse_relation} and \ref{fig:inverse_line}.
Figure~\ref{fig:inverse_line} sorts heads from low to high overlap and
plots overlap and AUROC under the same head order, making the inverse
tendency visually explicit. This surprising finding indicates that heads
with already strong spatial concentration are not necessarily the ones
most tied to final correctness. Based on this, we hypothesize:

\begin{center}
\fbox{\parbox{0.94\linewidth}{
\textbf{Hypothesis.}
Spatial reasoning can be improved more effectively by strengthening
heads with weak initial spatial grounding (bottom16), rather than further
increasing attention concentration in already strong spatial heads
(top16). This should lead to a more even spatial role distribution across
heads and better task performance.
}}
\end{center}

\section{Our Method: Function-Aware Spatial Attention Reallocation}
\textbf{Experiment settings.}
Unless otherwise specified, this section follows the same evaluation
setup as Section 3: \texttt{llava1.5}, the AdaptVis protocol, and
matched 100-sample splits for cross-method comparison. We evaluate on
four benchmarks with increasing relational difficulty:
\texttt{COCO\_QA\_one\_obj} and \texttt{VG\_QA\_one\_obj}
(single-object relation queries), and \texttt{COCO\_QA\_two\_obj} and
\texttt{VG\_QA\_two\_obj} (two-object relation queries). This split
allows us to check whether a method is robust only on simpler one-object
questions or still effective when relational interference increases.

\textbf{Naive idea.}
Our first idea was straightforward: directly boost attention inside the
YOLO box. If a region detector marks the target object, increasing
weight in that region seems likely to help spatial reasoning. In
practice, this direct strategy is unstable and often hurts semantic
consistency; detailed quantitative results are provided in
Appendix~\ref{sec:appendix_naive_eval}.

\subsection{Margin-Guided Continuous Reallocation}
To address the failure mode above, we replace hard region forcing with a
confidence-aware and smooth reallocation mechanism.

\textbf{(i) Margin-guided continuous control.}
At decoding step $t$, let $p_{(1),t}$ and $p_{(2),t}$ be top-1 and top-2 token probabilities. We define confidence margin
\begin{equation}
  m_t = p_{(1),t} - p_{(2),t}.
\end{equation}
Intervention strength is then set continuously by
\begin{equation}
  \alpha_t = \alpha_{\min} + (\alpha_{\max} - \alpha_{\min})\cdot \sigma\!\left(\beta(m_t-\tau)\right),
\end{equation}
where $\beta$ controls slope and $\tau$ controls the transition point.

\textbf{(ii) Attention reweighting without hard clipping.}
Given image-side attention $A_t$, we apply
\begin{equation}
  \tilde{A}_t = \mathrm{Normalize}\!\left(A_t^{\alpha_t}\right),
\end{equation}
which is the core execution step of our method: the confidence-controlled
exponent $\alpha_t$ is converted into a smooth redistribution of
attention mass over image patches. In practice, $\alpha_t>1$ sharpens
attention to concentrate evidence, while smaller values keep attention
diffuse, avoiding abrupt region-level clipping.

weaker initial spatial grounding absorb more spatial mass after
intervention, making spatial responsibility less concentrated in a few
heads. In parallel, both datasets improve in accuracy (+6.0 on
Control\_A, +5.0 on Control\_B), supporting the claim that our method
improves task performance by redistributing spatial reasoning load across
heads rather than only amplifying spatial heads that are already strong
in spatial reasoning.
Figure~\ref{fig:control_a_overlap_gain} gives a qualitative before/after
view at head level. Under the same normalization, warmer colors indicate
higher attention. The example suggests that our method can strongly
increase spatial focus for some initially weak heads while slightly
reducing concentration on some initially strong heads.

Group-level AUROC changes are mixed and not uniformly monotonic. Even
with this instability, accuracy improves on both Control datasets, which
suggests a redistribution pattern consistent with improved task
performance.

To sum up, our method improves weaker bottom16 heads and makes spatial reasoning roles
more evenly distributed, instead of relying on further strengthening the
already strong top16 heads to obtain better reasoning performance.

\section{Related Work}

\textbf{VLM attention reallocation.}

Recently broader attention pattern findings emerged in language models,
including middle-context underuse and attention-sink behavior
\citep{liu2024lost,yu2024unveiling}, and by training-free
intervention methods for attention control and steering
\citep{li2026spectral,zheng2024laser}.
AdaptVis is the closest prior work to ours, which shows that
intervening on image-token attention can improve spatial relational
prediction in VLMs \citep{chen2024adaptvis}. Our method follows this line but
shifts the question from ``how strong should global reweighting be'' to
``which heads should be reweighted.'' 
Compared with these methods, we target VLM spatial reasoning without manual segment
specification and use head-wise overlap/AUROC analysis to guide
reallocation.

\textbf{YOLO-grounded localization for analysis.}
Object detectors such as YOLO are widely used as practical region
proxies in mechanism studies. We adopt YOLO boxes to build a concrete
spatial mask and measure whether head attention falls into
detector-supported object regions, enabling sample-level and
dataset-level overlap diagnostics \citep{redmon2016you}.

\textbf{AUROC-based functional ranking.}
Attention concentration alone does not indicate functional usefulness.
We therefore connect per-head overlap to final correctness through
AUROC, producing a head-level functional ranking. This analysis reveals
that strongly concentrated spatial heads are not always the heads most
associated with correct answers, which directly motivates our
hypothesis-driven reallocation design \citep{fawcett2006introduction}.

\section{Conclusion}
We present an analysis-driven pipeline for head-level spatial reasoning:
diagnose head behavior, design intervention, and validate mechanism and
performance jointly. The central hypothesis of this work is that
improving spatial reasoning should not primarily rely on heads that are
already spatially concentrated, but should instead unlock heads with
weaker current spatial focus and higher latent functional potential. Our
results support this hypothesis. We observe a strong inverse relation
between overlap and AUROC across heads, and we further show that
accuracy gains are accompanied by systematic redistribution of spatial
mass from initially strong heads to weaker head groups. Overall, the
proposed margin-guided continuous method improves spatial reasoning while
providing mechanism-level evidence for why the improvement happens.

\section{Limitation}
First, we do not yet have a robust intervention that directly and
independently optimizes individual heads in a fully controllable manner.
Failure-case evidence is provided in Appendix~A.2. In practice, our best
results still come from an AdaptVis-based optimization over all heads,
which then leads to measurable head-level improvement.

Second, due to time constraints, we have not yet evaluated the full
pipeline on Qwen-series VLMs; current experiments are centered on
LLaVA-1.5.

Third, we originally planned to run the same YOLO-position analysis on
the other four datasets, but those images are much more cluttered, and
the match quality between YOLO boxes and question-target objects is
often unreliable. We therefore relied on cleaner controlled data for
mechanism analysis. A stronger validation on more complex datasets is an
important next step.

\section{Failure Case Discussion}
\subsection{Top-patch flattening}
An additional variant we explored is \emph{top-patch flattening inside
YOLO regions}. For patches whose centers fall inside the YOLO-expanded
box mask, we first select the top-$\beta$ fraction by attention score
(we use $\beta=\frac{1}{5}$), then force them toward a shared high level
($0.7A_{\max}$):
\begin{equation}
  \hat{A}_t(i)=
  \begin{cases}
    0.7A_{\max}, & i\in\mathcal{P}^{\text{box}}_{\beta},\\
    \tilde{A}_t(i), & \text{otherwise}.
  \end{cases}
\end{equation}
Although this variant was intended to stabilize redistribution, it often
over-flattened critical contrast and degraded answer quality
(especially on two-object settings) in our current runs. The key lesson
is that directly flattening high-attention patches weakens useful
relative attention structure across regions. We therefore treat
top-patch flattening as a failure-case direction rather than part of the
main method.

makes object-level competition less discriminative. This is consistent
with our design choice to use margin-guided continuous reweighting
instead of hard flattening.

\subsection{YOLO-based Naive Idea}
\label{sec:appendix_naive_eval}
We also tested a direct naive strategy that boosts in-box attention
without margin-based uncertainty control or continuous reweighting.
Concretely, let $A_{\max}$ be the global maximum attention value of the
current map; for every location inside the YOLO boxes, we directly set the
attention to a uniform level $\alpha A_{\max}$ (i.e., in-box flattening
to an $\alpha$-scaled global peak). We sweep
$\alpha\in\{0.5,0.6,0.7,0.8,0.9,1.0\}$ and report the best naive result
for each dataset in Table~\ref{tab:appendix_naive_eval_results}. For the
top-$\beta$ flattening variant, we additionally enumerate
$\beta\in\{0.1,0.2,0.3,0.4\}$ (top 10\%, 20\%, 30\%, and 40\%) and
report representative outcomes in the qualitative comparison.

After this sweep, the naive intervention does not provide consistent
gains over our best method and is often clearly worse.
Together with the top-flat failure above, this result suggests that
simply forcing attention to become flatter or uniformly larger inside
the box is not sufficient. A better intervention should preserve
relative attention information, because these relative differences likely
encode object relations needed for reasoning.
